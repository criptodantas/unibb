## 🚀 Instalação e Uso

### **Pré-requisitos**

- Python 3.10 ou superior
- Node.js (opcional, para servidor local)
- Conta OpenAI com API key
- (Opcional) Dispositivo Android com modo desenvolvedor para automação mobile

### **Passo 1: Clone o Repositório**

```bash
git clone https://github.com/seu-usuario/unibb-ai-assistant.git
cd unibb-ai-assistant
```

### **Passo 2: Instalar Dependências Python**

```bash
# Instalar dependências
pip install -r requirements.txt

# Instalar navegador do Playwright
playwright install chromium
```

### **Passo 3: Configurar Credenciais**

Copie o arquivo de exemplo e configure:

```bash
cp .env.example .env
```

Edite `.env` com suas credenciais:

```env
UNIBB_LOGIN=F1234567
UNIBB_SENHA=sua_senha_unibb
OPENAI_API_KEY=sk-proj-sua_chave_openai
```

### **Passo 4: Abrir Dashboard Web**

Simplesmente abra o arquivo `index.html` no navegador:

```bash
# Linux/Mac
open index.html

# Windows
start index.html

# Ou use um servidor local
python -m http.server 8000
# Acesse: http://localhost:8000
```

### **Passo 5: Executar Bot Python**

Para automação completa via linha de comando:

```bash
python unibb_bot.py
```

## 🐛 Problemas Conhecidos e Soluções

### **Problema: Login falha**
**Solução**: Verifique credenciais no `.env` e se o site UniBB está acessível

### **Problema: IA retorna respostas incorretas**
**Solução**: 
- Verifique se a API key OpenAI está válida
- Aumente o contexto fornecido (processar mais PDFs)
- Use GPT-4o ao invés de GPT-3.5

### **Problema: Playwright não encontra elementos**
**Solução**: 
- A estrutura HTML da UniBB pode ter mudado
- Ajuste os seletores em `unibb_bot.py`
- Ative modo headless=False para debug visual

### **Problema: App mobile não conecta**
**Solução**:
- Confirme que modo desenvolvedor está ativo
- Execute `adb devices` para verificar conexão
- Reinstale drivers USB do dispositivo
